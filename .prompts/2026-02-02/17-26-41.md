## User Request (Chinese)

写一个 ModelManager的class，用来查询，下载，删除，管理本地模型，支持设置保存目录，Pipeline.ts从ModelManager获取嵌入和重排模型(注意模型下载到一版可能超时，看能不能断点下载，或是验证模型不完整之后重新下载，需要为每个本地模型添加一个是否可用的状态变量)

添加一个 Cli的class，让Polywise和ModelManager支持命令行调用

基于 https://www.npmjs.com/package/commander

## Answers to Clarifying Questions

- **ModelManager 需要支持从哪些来源下载模型？**=HuggingFace Hub
- **CLI 需要支持哪些命令？**=Model management + Polywise basic ops
- **断点下载和失败重试的实现方式？**=Hybrid approach

## Implementation Plan

1. Add dependencies: `commander`, `@huggingface/hub`
2. Create `types/model.ts` with `ModelStatus`, `LocalModel`, `ModelDownloadProgress`, `ModelManagerArgs`
3. Create `ModelManager.ts` with methods: init, listModels, getModel, downloadModel (with resume), deleteModel, verifyModel, setModelsDir, getModelStatus
4. Modify `Pipeline.ts` to use `ModelManager`
5. Create `cli/index.ts` as CLI entry (no separate PolywiseCLI file)
6. Create CLI command files in `cli/commands/`
7. Update `index.ts` exports
8. Update `agentmap.md`

## CLI Commands

```bash
# Models
polywise models list
polywise models download <model_id>
polywise models delete <model_id>
polywise models status <model_id>
polywise models verify <model_id>

# Polywise
polywise search <query>
polywise add-article <title> <content>
polywise add-node <label> <x> <y>
polywise connect <source_id> <target_id>
polywise stats

# Config
polywise config set models-dir <path>
polywise config show
```
