# Opencode Polywise 插件

[English Documentation](./README.md)

## 简介

`@polywise/opencode-plugin` **Polywise** 知识引擎为opencode提供的插件，它为您的 AI 编程助手提供项目级别的记忆和上下文感知能力。

利用 Polywise 的向量数据库和知识图谱能力，该插件可以自动：

1.    **检索上下文**：根据您当前的查询，将相关的项目知识（文档、代码片段、决策）注入到 AI 的提示词中。
2.    **保存记忆**：自动将 AI 的回答和见解持久化到项目专属的知识库中，以便未来调用。

这确保了您的 AI 助手能够在不同的会话中“记住”架构决策、代码模式和项目特定的规则。

## 特性

- **项目隔离**：数据存储在项目目录下的 `.polywise/` 文件夹中。每个项目都有自己隔离的记忆，通过唯一的项目 ID 进行标记。
- **自动检索 (RAG)**：在 AI 回答之前，插件会向 Polywise 查询相关上下文，并将其附加到系统提示词中。
- **自动持久化**：在每轮对话结束后，AI 的输出会被保存到知识图谱中，从而构建项目演进的长期记忆。
- **元数据支持**：捕获并存储元数据，如文件链接、消息角色和时间戳。

## 安装

要在 OpenCode 中使用此插件：

1.    **安装包**：
      确保 `@polywise/opencode-plugin` 已构建并在您的工作区中可用，或者已安装在您的项目中。

2.    **配置 OpenCode**：
      将插件添加到您的 `opencode.json`（项目级）或 `~/.config/opencode/opencode.json`（全局级）中。

      ```json
      {
      	"plugins": ["@polywise/opencode-plugin"]
      }
      ```

      _注意：如果您是在本地链接它，请调整包名称路径。_

## 使用方法

安装完成后，插件将在后台自动工作。

1.    **正常对话**：当您在 OpenCode 中提问时，如果找到相关信息，您可能会看到“Project Memory”部分被注入到提示词上下文中。
2.    **记忆增长**：随着您与 AI 互动的增加，本地数据库 (`.polywise/`) 会不断增长，使得未来的回答更加准确并具有上下文感知能力。
