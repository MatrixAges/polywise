# Agent Private Contents

This page explains the English-first structure for agent-private content handling. It is meant to clarify how non-user-facing context can remain isolated while still supporting reasoning.

## What this page will cover

- The boundary between shared and private content
- Storage and retrieval expectations for agent-only material
- Safety considerations when private context affects output

## Current status

The route is now stable and can be expanded with policy, storage, and workflow details later.
