# Intro

Polywise is an open source agentic content system. It’s available as a terminal-based interface, web app, or desktop application.

![linkcase](/images/linkcase.png)

## Install

**Prerequisites**

To use Polywise in your terminal, you’ll need:

### Required

- [Nodejs](https://nodejs.org): cross-platform javaScript runtime

### Optional

- [Agent Browser](https://agent-browser.dev/installation): native browser automation provider for agents
- [OpenCli](https://opencli.info/docs/guide/installation.html): turns any website or Electron app into a command-line interface
- [Crawl4ai](https://docs.crawl4ai.com/core/cli/): open-Source LLM-Friendly Web Crawler & Scraper
- [Dokobot](https://dokobot.ai/zh-CN/help/install-cli): local browser agent CLI that can read through your logged-in Chrome session
- [Jina Reader](https://jina.ai/en-US/reader/): remote fallback content reader used as `r.jina.ai` / `s.jina.ai`

## Why Polywise

**When knowledge and experience become less critical, from where does human value derive? How is human uniqueness manifested?**

It stems from the "weight" of one's thinking—specifically, from the unique architecture of each individual's neural networks.
Redefining human value: in the era of AI, human worth lies in the plasticity of one's intelligence—that is, in how one leverages AI to construct a personal digital avatar.

As long as the concept of private ownership persists, the capacity for human individuality retains its value, serving as a unique and precious personal asset. However, these capabilities must be cultivated and crystallized through sustained interaction with AI. Polywise integrates the user and the AI ​​into a single, evolving, and iterative intelligent entity—thereby amplifying human value, rather than replacing it.

To address the issues of the knowledge explosion and information overload characteristic of the AI ​​era, we leverage AI's powerful inductive capabilities to perform "post-thinking"—thereby freeing up human cognitive resources for matters of greater importance.
