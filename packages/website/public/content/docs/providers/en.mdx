# Providers

This page is the English-first home for model provider documentation. It is intended to explain which model providers Polywise supports, how their capabilities differ, and what tradeoffs matter when choosing between them.

## What this page will cover

- Capability differences between supported model providers
- Model availability, behavior, and configuration tradeoffs
- Selection criteria for common deployment scenarios
